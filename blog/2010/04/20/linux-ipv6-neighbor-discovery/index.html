Linux IPv6 Neighbor Discovery

Apr 20th, 2010 4:37 pm

Using ping6 and ip neigh it’s possible to discover other IPv6 hosts on your local network using the link-local addresses. First, use ping6 to ping all hosts on the local network using the IPv6 link-local address

ping6 -c 2 -I eth0 ff02::1

Then, use ip neigh to show a list of IPv6 enable hosts on the local network

ip -f inet6 neigh
